Oque é listas ordenadas?

Lista ordenadas no html, são lista que tem como objetivo definir oque vem primeiro, como po exemplo ( acordar, escovar os dentes, comer etc... )

para tranformar isso no html use a tag <ol> junto com a tag <li>, oque significa ol? fique com o scroll do mouse aqui em cima para saber oque é OL

Vejamos como fica essa lista ordenada na pratica:

  1. acordar
  2. tomar cafe
  3. escovar os dentes
  4. ir para escola
  5. fazer lições
  6. ir para casa
  7. fazer comida
  8. deixar noticias em dia
  9. fazer ranking

temos tambem alguns tipos de referencias de lista ordenadas, sendo elas numericas, alfabetica e romanos!

  1. Esse
  2. é o teste
  3. para a
  4. lista
  5. alfabetica
  1. esse
  2. é o teste
  3. para a lista
  4. romanas

lista não ordenadas!

as lista não ordenadas são aquelas na qual nao tem objetivo de qual fazer primeiro, mas ainda sim é uma lista
utilizamos a tag <ul> para chamar essa lista não ordenada veja como ela fica na pratica:

  • Este é um exemplo
  • de listas
  • não ordenadas
